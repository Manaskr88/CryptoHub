# 📈 Crypto Tracker Application

🔗 **Live Demo:** https://manascryptohub.netlify.app/  
💻 **GitHub Repository:** https://github.com/Manaskr88/crypto  

---

## 📖 Overview

Crypto Tracker is a real-time cryptocurrency tracking web application built using React.js.  
It fetches live market data from the CoinGecko API and displays dynamic price trends for 20+ assets.

The application focuses on performance optimization, responsive UI design, and seamless live updates.

---

## 🚀 Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- CSS

### API
- CoinGecko REST API

### Deployment
- Netlify

---

## ✨ Key Features

- Real-time price tracking for 20+ cryptocurrencies  
- Dynamic chart visualization  
- Multi-currency support  
- Optimized data refresh for faster updates

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/Manaskr88/crypto
```

### 2️⃣ Navigate to project directory
```bash
cd crypto
```

### 3️⃣ Install dependencies
```bash
npm install
```

### 4️⃣ Start the development server
```bash
npm start
```

The application will run on:

```
http://localhost:3000/
```
